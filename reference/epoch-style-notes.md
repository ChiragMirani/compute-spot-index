# Epoch Style Notes

Reference file: `reference/epoch-home.html`

Use Epoch as inspiration, not as a copied design.

- Research-first page: headline states the public-interest purpose, then data modules carry the page.
- Warm paper background with off-white data surfaces.
- Low-radius rectangular sections, thin borders, visible grid structure.
- Large trend/stat numbers with short labels and confidence/caveat text nearby.
- Navigation and category labels are small, calm, and utilitarian.
- Tables/data explorers should feel auditable: source, timestamp, assumptions, and methodology should be visible.
- Avoid decorative gradients or marketing cards; use restrained blocks, separators, and compact labels.
- Language should emphasize evidence, estimates, and uncertainty.
